●概要：どんなアプリなのか？

・イメージする言葉、単語から子供の名前候補を検索するアプリケーション
・歴史上の人物名検索やキーワードから過去の偉人の経歴や名前を表示出来る
・苗字と名前の画数から来る相性の判断も行う事が出来る

//=================================

//=================================

タスク：
1. 開発環境（Android SDK）のセットアップ
2. HTTPのPOSTまたはGETでリクエスト（キーワード）を送信できること
3. サーバからのレスポンスを画面に表示できること

2, 3に関してはサーバ側の準備がまだなので、HTTPの操作ができればとりあえず
